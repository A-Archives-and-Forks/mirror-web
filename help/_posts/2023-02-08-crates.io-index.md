## Rust crates.io 稀疏索引镜像使用帮助

编辑 `~/.cargo/config` 文件，添加以下内容：

```
[source.crates-io]
replace-with = 'tuna'

[source.tuna]
registry = "sparse+https://{{ site.hostname }}/crates.io-index/"
```

注：`sparse+` 表示在使用稀疏索引，链接末尾的 `/` 不能缺少。

截至目前，可以通过 `cargo +nightly -Z sparse-registry update` 使用稀疏索引。
